# Documentation Index

Welcome to your dotfiles documentation. This is a v2.2.0 productivity system with 66 scripts, 21 libraries, 13 AI dispatchers, a Cyborg Lab content agent, AI coaching, and brain-fog autopilot mode.

---

## 5-Minute Orientation

```bash
dotfiles-check              # Validate your environment
startday                    # Morning briefing + AI coaching
todo top && status          # Work from your top queue
goodevening                 # Evening reflection + backup
```

Brain-fog day? Just type `ap` and follow the A-E prompts.
