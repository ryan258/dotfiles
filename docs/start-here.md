# Start Here

This is a derived quickstart. Canonical behavior lives in `../CLAUDE.md`.

## 5-Minute Orientation

1. Validate the environment:
```bash
dotfiles-check
```
2. Run the morning briefing:
```bash
startday
```
3. Work from the top queue:
```bash
todo top
status
```
4. Close the day with reflection + backup:
```bash
goodevening
```

## Core Commands

- `startday` - morning context + AI coach briefing
- `status` - mid-day context reset
- `todo add "task"` / `todo top` / `todo done <id>`
- `journal add "note"`
- `health energy <1-10>` / `health fog <1-10>`
- `focus set "one clear outcome"`
- `goodevening` - evening reflective closeout
